## About

My name's Chris and I converted a RAM Promaster into a campervan a few years ago. My time in the van is usually spent rock climbing, traveling, and adventuring outdoors, but I'm also always working on projects to improve the van.

The most recent project I've been working on has been a home automation system for the van. I'm using a Raspberry Pi running **Home Assistant** as the central hub. This is an ongoing work in progress, but I'm already getting a lot of use out of the system. I created this site for my own documentation and for others who want to implement something similar.
